Keyboard Shortcuts

Ctrl and z
Use when you make a mistake and want to undo your last step
Ctrl and y
Use when you undo a step, but now wish to revert back
Ctrl and a
Use to select the entire document
Ctrl and c
Use to copy all that is selected to the clipboard
Ctrl and x
Use to copy all that is selected to the clipbaord while also removing it from its current location
Ctrl and v
Use to paste all that you previously copied to the clipboard
Ctrl and s
Use to save your work on the document currently open
Ctrl and f
Use to search the page you are working on for a keyword
